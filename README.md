# Social Media Post Composer

A responsive multi-platform post composer built for **Experiment 1.1.1**.  
It allows users to create a post, attach media, select one or more social platforms, and receive real-time validation based on platform-specific rules.

## Live Demo

[Open Demo](https://yogesh4921.github.io/Social-Media-Post-Composer/)

## Aim

To design and develop a dynamic post composer interface supporting multiple platforms with constraint validation.

## Objectives

- Understand multi-platform content handling
- Implement real-time validation mechanisms
- Design responsive and user-friendly UI components

## Features

- Multi-platform selection
- Real-time character and media validation
- Platform-specific feedback and warnings
- Media attachment support
- Live preview panel for different platforms
- Theme toggle for light/dark mode
- Responsive UI for desktop and mobile

## How It Works

1. Select one or more social media platforms.
2. Enter post content in the composer.
3. Attach media files if needed.
4. The system validates the content in real time.
5. Users see warnings, errors, and preview layouts instantly.
6. Publishing is enabled only when selected platform rules are satisfied.
